# Ataxx - Ανάπτυξη Παιχνιδιού για Εργασία Σχολής

Το **Ataxx** είναι ένα παιχνίδι στρατηγικής για δύο παίκτες που παίζεται σε πλέγμα 7x7. Αυτή η υλοποίηση αποτελεί μέρος μιας εργασίας σχολής, με σκοπό την πλήρη ανάπτυξη του παιχνιδιού με υποστήριξη Human-Human λειτουργίας και βασικό GUI μέσω του browser.

## Χαρακτηριστικά Έργου
- **Πλήρης Υλοποίηση Παιχνιδιού**: Το παιχνίδι περιλαμβάνει όλους τους κανόνες όπως ορίζονται στο κλασικό Ataxx.
- **Human-Human Λειτουργία**: Δύο παίκτες μπορούν να παίξουν εναλλάξ στον ίδιο browser.
- **Βασικό GUI**: Μια απλή και λειτουργική διεπαφή χρήστη για αλληλεπίδραση μέσα από τον browser.

## Τεχνικές Προδιαγραφές
- **Γλώσσα Προγραμματισμού**: JavaScript (για το front-end), Python/Node.js (για back-end αν απαιτείται).
- **Πλαίσιο Ανάπτυξης**: HTML, CSS, JavaScript.
- **Περιβάλλον Ανάπτυξης**: Οποιοσδήποτε σύγχρονος browser.

## Οδηγίες Εγκατάστασης
1. Κλωνοποιήστε το αποθετήριο:
   ```bash
   git clone https://github.com/your-repo/ataxx.git
   ```

2. Εγκαταστήστε τις εξαρτήσεις (αν υπάρχει back-end):
   ```bash
   npm install
   ```

3. Εκκινήστε την εφαρμογή:
   ```bash
   npm start
   ```

4. Ανοίξτε τον browser σας στη διεύθυνση:
   ```
   http://localhost:3000
   ```

## Κανόνες Παιχνιδιού

### Σκοπός του Παιχνιδιού
Ο στόχος είναι να αποκτήσετε περισσότερα κομμάτια από τον αντίπαλο στο τέλος του παιχνιδιού.

### Διάταξη και Έναρξη
- Το παιχνίδι παίζεται σε ένα πλέγμα 7x7.
- Κάθε παίκτης ξεκινά με δύο κομμάτια που τοποθετούνται στις διαγώνια αντίθετες γωνίες του πλέγματος.
- Οι παίκτες παίζουν εναλλάξ.

### Κινήσεις

#### 1. Αναπαραγωγή (Replication)
Ένα κομμάτι μπορεί να "αναπαραχθεί" σε ένα κενό τετράγωνο που βρίσκεται δίπλα του (σε απόσταση μίας θέσης). Όταν γίνει η κίνηση, δημιουργείται ένα νέο κομμάτι του παίκτη στο τετράγωνο αυτό.

#### 2. Άλμα (Jump)
Ένα κομμάτι μπορεί να κινηθεί σε ένα κενό τετράγωνο που βρίσκεται σε απόσταση δύο θέσεων (οριζόντια, κάθετα ή διαγώνια). Με αυτή την κίνηση, το αρχικό κομμάτι μεταφέρεται στη νέα θέση, χωρίς να δημιουργείται νέο κομμάτι.

### Μετατροπή (Capture)
Όταν ένα κομμάτι μετακινείται (είτε μέσω αναπαραγωγής είτε μέσω άλματος), όλα τα αντίπαλα κομμάτια που βρίσκονται σε γειτονικά τετράγωνα μετατρέπονται σε κομμάτια του παίκτη που έκανε την κίνηση.

### Τέλος Παιχνιδιού
Το παιχνίδι τελειώνει όταν:
- Δεν υπάρχουν περισσότερες δυνατές κινήσεις για τους παίκτες.
- Το πλέγμα γεμίζει.

Ο νικητής είναι ο παίκτης με τα περισσότερα κομμάτια στο πλέγμα.

## Σημειώσεις για την Υλοποίηση
- Η διεπαφή είναι σχεδιασμένη για απλή χρήση, με επιλογές για κίνηση των κομματιών μέσω κλικ και επιλογής.
- Η λογική του παιχνιδιού βασίζεται σε χειρισμό των κανόνων σε JavaScript για την άμεση ανταπόκριση στον browser.
